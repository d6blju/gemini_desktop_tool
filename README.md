# Gemini Desktop Tool

> **Note**: This project was entirely written by AI (Google Gemini) under the guidance of a human user.

## Overview

Gemini Desktop Tool is a productivity utility that seamlessly integrates Google Gemini into your desktop workflow. It allows you to quickly capture text and images from any application and send them directly to Gemini for analysis, summarization, or creative writing.

## Features

-   **Instant Access**: Trigger the tool from anywhere using the global hotkey (`Ctrl+Alt+G`).
-   **Smart Context Capture**: Automatically captures the selected text from your active window when triggered.
-   **Visual Input**:
    -   Take screenshots of specific screen regions.
    -   Paste images directly from your clipboard.
    -   Support for multiple images with preview and management.
-   **Seamless Automation**:
    -   Automatically opens or focuses the Gemini web interface.
    -   Pastes your instructions, captured text, and images.
    -   Sends the prompt for you.
-   **Session Management**: Option to reuse the existing Gemini window to maintain context.
-   **History Support**: Remembers your recent instructions for quick reuse.
-   **Configurable**: Customize hotkeys, delays, and other settings via `config.json`.
-   **Unobtrusive**: Runs quietly in the system tray.

## How to Use

1.  **Run the Tool**: Launch `GeminiTool.exe`. You will see a "G" icon in your system tray.
2.  **Select Text (Optional)**: Highlight text in any application (browser, IDE, document).
3.  **Trigger**: Press `Ctrl+Alt+G`.

## Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| `Ctrl+Alt+G` | Open Gemini Tool popup |
| `Ctrl+Alt+Q` | Exit the application |
| `Ctrl+Enter` | Send to Gemini (in popup) |
| `Esc` | Close popup |
| Right-click on image | Remove image |

## Configuration

Edit `config.json` to customize the tool:

```json
{
    "hotkey": "ctrl+alt+g",
    "exit_hotkey": "ctrl+alt+q",
    "gemini_url": "https://gemini.google.com/app",
    "paste_delay_new": 6.0,
    "paste_delay_reuse": 1.5,
    "max_history": 10
}
```
